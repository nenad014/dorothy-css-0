Buttons

Button in different colors, sizes and states

The button is an essential element of any design. It's meant to look and behave as an interactive element of your page.

The .btn class can be used on:

  • <a> anchor links
  • <button> form buttons
  • <input type="submit"> submit inputs
  • <input type="reset"> reset inputs
Anchor

Colors

Standard

The button is available in all the defined colors



Outline

Dorothy.css also provides outline/bordered buttons


Social networks

Dorothy.css also provides buttons with social network colors

* For using icons we are recommending to import latest version of Font Awesome


Contextual colors

In the end, but also important, Dorothy.css provides us contextual colors like on Bootstrap. They can be also outlined



Sizes

The button comes in 3 different sizes:

  • small
  • medium (default)
  • large

Use the .btn-l class for large buttons or .btn-s class for small buttons:


Other types of buttons

Dorothy.css provides few more styles of buttons:

  • circle buttons .btn-circle
  • square buttons .btn-square
  • pill buttons .btn-pill
  • full width buttons .btn-fullwidth
