Grid

We are using a standard 12 column fluid responsive grid system. The grid helps you layout your page in an ordered, easy fashion.


Container

.container class is one of the most important classes in Dorothy.CSS. It allows you to center your page content. .container class has a default 32px left and right padding, and no top or bottom padding.

.container class is the perfect to use for all HTML container elements such as div, article, section, header, footer, ...

To add a container just put your content inside a div tag with a container class. Here's an example of how your page might be set up.


Panel

.panel class is similar to the .container class.

.panel class adds a 16px top and botom margin and a 32px left and right padding to any HTML element.
